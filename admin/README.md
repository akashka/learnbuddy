# Tuition Platform – Admin

Admin dashboard for managing teachers, parents, students, enrollments, masters, AI data, and schedules.

## Setup

```bash
npm install
cp .env.example .env
# Set VITE_API_BASE_URL to your backend URL (e.g. http://localhost:3001)
npm run dev
```

Runs at http://localhost:5174 by default.

## Build & Deploy

```bash
npm run build
```

Deploy the `dist/` folder to Vercel, Netlify, or any static host.
