## [0.1.0] — Initial Release

### API

- **Auth:** Login, register, OTP, JWT, logout, email verification
- **Parent:** Profile, students (CRUD), wishlist, checkout, payment, classes, reviews
- **Teacher:** Profile, batches, classes, payments, exams, agreements
- **Student:** Profile, courses, classes, exams
- **Admin:** Users, teachers, parents, students, enrollments, masters, AI usage logs, AI review requests, security incidents
- **Privacy:** Dashboard, data export
- **Account:** Delete request (OTP), delete confirm

### AI Service

- Exam generation (student, teacher qualification)
- Exam evaluation (MCQ, short answer, drawing)
- Study material generation
- Classroom and exam monitoring (video frames)
- Doubt answering
- Document verification

### Product

- Parent marketplace, enrollment, checkout
- Teacher batches, class scheduling
- Student exams, study materials
- AI monitoring with consent
- Human review for AI decisions
- Privacy dashboard and data export
- DPDP-oriented consent logging (child data, AI monitoring)
